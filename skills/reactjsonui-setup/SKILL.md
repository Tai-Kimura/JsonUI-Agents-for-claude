# ReactJsonUI Setup

## App Configuration (REQUIRED)

Before starting setup, read the app configuration files in `<app_config_path>` (provided by parent agent):

1. Read all files in `<app_config_path>` to understand project requirements:
   - **Library dependencies and versions** → Use for `package.json` (e.g., ReactJsonUI version, Next.js version, third-party npm packages)
   - **Import settings** → Determine which modules to import in components and pages
   - **Platform requirements** → Node.js version, TypeScript configuration
   - App name, etc. → Use for page titles and component naming
2. Apply these settings throughout the setup process (especially dependency versions in `package.json`)

## NEVER CREATE FILES MANUALLY

**DO NOT create any files yourself.** All files are generated by commands:

- JSON files → `g view`, `g partial`, `g collection` commands
- React components → `build` command
- Config files → `init` command

## Checklist

- [ ] Step 1: Project root confirmed (package.json exists)
- [ ] Step 2: rjui_tools installed
- [ ] Step 3: rjui.config.json created
- [ ] Step 4: setup command executed
- [ ] Step 5: Splash view generated
- [ ] Step 6: rjui build executed
- [ ] Step 7: Final verification passed

---

## Step 1: Go to project root
```bash
cd <project_directory>  # Provided by setup agent
ls package.json         # VERIFY: Must show package.json
```

## Step 2: Copy rjui_tools to project
```bash
cp -r <jsonui_cli_path>/rjui_tools ./
ls rjui_tools/bin/rjui  # VERIFY: File must exist
```

## Step 3: Initialize
```bash
./rjui_tools/bin/rjui init
ls rjui.config.json  # VERIFY: File must exist
```

## Step 4: Verify and Edit rjui.config.json

**Required checks:**
1. `layouts_directory` - Usually `layouts`
2. `components_directory` - Usually `components/generated`
3. `styles_directory` - Usually `styles`

## Step 5: Run setup
```bash
./rjui_tools/bin/rjui setup
```

## Step 6: Generate Splash view
```bash
./rjui_tools/bin/rjui g view Splash --root
```

## Step 7: Build to generate React components
```bash
./rjui_tools/bin/rjui build
```

## Step 8: Final Verification

Verify these files exist:
1. `rjui_tools/bin/rjui`
2. `rjui.config.json`
3. `{layouts_directory}/Splash/`
4. `{components_directory}/SplashView.tsx`

## Hotloader Command
```bash
./rjui_tools/bin/rjui hotload
```
